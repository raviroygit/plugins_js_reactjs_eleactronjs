<===================================== Setup Splashscreen ============================>

1) Create directory resources/splash in root of project.

2) Add splash.css and splash.html file of custom splashscreen .

3) Import plugin like below mentioned ðŸ‘‡.

4) Add startSplashscreen and stopSplashscreen inside ready-to-show callback functon ðŸ‘‡.
